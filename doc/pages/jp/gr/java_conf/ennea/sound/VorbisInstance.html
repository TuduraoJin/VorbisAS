class jp.gr.java_conf.ennea.sound.VorbisInstance

Available on all platforms

Vorbis Instance supported OggVorbis playback control and channel management. you can use easily normal playback, multiple fx playback, and fadeIn/Out.

@author Tudurao Jin

Float of times to loop this sound. 0 or 1 is no loop. (play Once) 2...X is play X times. -1 is infinitely loop.

Loops remaining, this will auto-decrement each time the sound loops. It will equal 0 when the sound is completed, or not looping at all. It will equal -1 if the sound is looping infinitely.
